# Plan: Записать Lorem Ipsum в test.md

## Context

Пользователь хочет заполнить `test.md` классическим Lorem Ipsum — для проверки рендеринга текста в плагине.

## Целевой файл

- `test.md` — корень проекта

## Содержимое для записи

```markdown
# Lorem Ipsum

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

## De Finibus Bonorum et Malorum

Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

### Sectio Prima

Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur.

### Sectio Secunda

Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.

## Cicero, 45 BC

> "Neque porro quisquam est, qui dolorem ipsum quia dolor sit amet, consectetur, adipisci velit."
```

## Шаги реализации

1. Открыть файл `test.md`
2. Записать указанное содержимое

## Проверка

После выполнения открыть `test.md` в редакторе и убедиться, что Markdown корректно отображается.
